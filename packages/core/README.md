# jupyterlab-git-core

Core Git operations library for the [jupyterlab-git](https://github.com/jupyterlab/jupyterlab-git) JupyterLab extension.

This package provides the pure Python Git logic without tornado/jupyter-server dependencies.
It is intended to be used as a dependency of `jupyterlab-git`.

See the [main repository](https://github.com/jupyterlab/jupyterlab-git) for full documentation.
